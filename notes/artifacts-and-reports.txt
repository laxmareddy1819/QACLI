What's Playwright-ONLY
Feature	Playwright	Everyone Else
Screenshots	✅	❌
Videos	✅	❌
Traces	✅	❌
Retry count	✅	❌

So to answer your question: test results, errors, and status work for all frameworks. But screenshots, videos, and traces only work for Playwright — because only Playwright's JSON report includes attachment paths in a standardized way.

The other frameworks (Jest, Cucumber, pytest, Robot, Maven, .NET) don't have a built-in way to report artifact file locations in their output formats.
